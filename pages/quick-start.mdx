# Quick Start

<Steps>

### Deploy a Database

Using the [web console](https://console.weavedb.dev/), follow the instructions on [Deploying a Database](/get-started/deploy-database)

### Create a Collection

Using the [web console](https://console.weavedb.dev/), follow the instructions on [Creating a Collection](/get-started/create-collection)

### Install WeaveDB

<Tabs items={['Web', 'NodeJS']}>
  <Tab>
    **yarn**:
    ```bash copy
    yarn add weavedb-sdk
    ```
    **npm**:
    ```bash copy
    npm install weavedb-sdk
    ```
  </Tab>
  <Tab>
    **yarn**:
    ```bash copy
    yarn add weavedb-sdk-node
    ```
    **npm**:
    ```bash copy
    npm install weavedb-sdk-node
    ```
  </Tab>
</Tabs>

### Initialize WeaveDB

<Tabs items={['Web', 'NodeJS']}>
  <Tab>
  ```js copy
  import WeaveDB from "weavedb-sdk"

  const db = new WeaveDB({ contractTxId: YOUR_WEAVEDB_CONTRACT_TX_ID })
  await db.init()
  ```
  </Tab>
  <Tab>
  ```js copy
  import WeaveDB from "weavedb-sdk-node"

  const db = new WeaveDB({ contractTxId: YOUR_WEAVEDB_CONTRACT_TX_ID })
  await db.init()
  ```
  </Tab>
</Tabs>

### Add a document

```js copy
const personData = { name: "Bob", age: 20 }
const result = await db.add(personData, "your_collection_name")
```

### Get a collection

Use `get` to read a collection:

```js copy
const result = await db.get("your_collection_name")
```

And `cget` to return the metadata of the documents too:

```js copy
const result = await db.cget("collection_name")
```

</Steps>
