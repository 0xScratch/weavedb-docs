# Quick Start

<Steps>

### Deploy a Database

Using the [web console](https://console.weavedb.dev/), follow the instructions on [Deploying a Database](/web-console/deploy-database)

### Create a Collection

Using the [web console](https://console.weavedb.dev/), follow the instructions on [Creating a Collection](/web-console/create-collection)

### Install WeaveDB

Execute one of the following commands in the root directory of your project.

<Tabs items={['Web', 'NodeJS']}>
  <Tab>
    **yarn**:
    ```bash copy
    yarn add weavedb-sdk
    ```
    **npm**:
    ```bash copy
    npm install weavedb-sdk
    ```
  </Tab>
  <Tab>
    **yarn**:
    ```bash copy
    yarn add weavedb-sdk-node
    ```
    **npm**:
    ```bash copy
    npm install weavedb-sdk-node
    ```
  </Tab>
</Tabs>

### Initialize WeaveDB

<Tabs items={['Web', 'NodeJS']}>
  <Tab>
  ```js copy
  import WeaveDB from "weavedb-sdk"

  const db = new WeaveDB({ contractTxId: CONTRACT_TX_ID })
  await db.init()
  ```
  </Tab>
  <Tab>
  ```js copy
  import WeaveDB from "weavedb-sdk-node"

  const db = new WeaveDB({ contractTxId: CONTRACT_TX_ID })
  await db.init()
  ```
  </Tab>
</Tabs>

<Callout type="info" emoji="ðŸ’¡">
Replace `CONTRACT_TX_ID` with the `contractTxId` string value returned when deploying your database.
</Callout>

### Add a document

```js copy
/**
 * @param {Object} personData - JSON data for the new document
 * @param {string} collection_name - The collection where the document will be added
 */
const personData = { name: "Bob", age: 20 }
const tx = await db.add(personData, collection_name)
```

<Callout type="info" emoji="ðŸ’¡">
Replace `collection_name` with the string value name of your collection.
</Callout>

### Get a collection

```js copy
const result = await db.get(collection_name)
```

<Callout type="info" emoji="ðŸ’¡">
Replace `collection_name` with the string value name of your collection.
</Callout>

</Steps>
