# Create a Collection

Using the [web console](https://console.weavedb.dev/), navigate to the `Data Collections` section in the side menu. Next, click on the `+` icon, which will open a dialog window. Input the desired name for your collection into the `Collection ID` field. Finally, to confirm and create the collection, click on the `Add` button.
