# Clone the Repo

[WeaveDB](https://github.com/weavedb/weavedb) is a monorepo that consists of Warp contracts, SDK, REPL, a web console, Jots (Social App) and many helper scripts.

Open a terminal and run the following commands to clone the WeaveDB repository and install its dependencies.

<Tabs items={['yarn', 'npm']}>
  <Tab>
  ```bash copy
git clone https://github.com/weavedb/weavedb.git
cd weavedb
yarn
  ```
  </Tab>
  <Tab>
  ```bash copy
git clone https://github.com/weavedb/weavedb.git
cd weavedb
npm install
  ```
  </Tab>
</Tabs>
